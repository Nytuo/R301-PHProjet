# Comics Sans MS

A web application for managing and selling books with easy addition via the Google Books API and secure payments using Stripe.

## Features

- **Add Books via Google Books API:**  
  Easily add books to the application's database using the Google Books API, streamlining catalog management for administrators.

- **Admin Interface:**  
  Simple administrator interface for managing books, orders, and users.

- **Order and Pay with Stripe:**  
  Customers’ orders are handled securely with Stripe Checkout integration, providing a seamless payment experience.

- **Modern PHP Web Application:**  
  Built using modern PHP best practices.

## Book Management

- Add and manage books to the catalog using the built-in admin interface.
- Integration with the Google Books API allows automatic fetching of book details.

## Payment Integration

- Uses Stripe Checkout for processing secure payments.
- Stripe PHP SDK is included in `PHPPR/payment/stripe-php-10.3.0/`.
